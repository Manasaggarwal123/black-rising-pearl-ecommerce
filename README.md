# Black Rising Pearl E-Commerce

A modern jewellery e-commerce web application built using React.js and Tailwind CSS.

## Features

- Product Listing
- Search Functionality
- Category Filter
- Price Filter
- Shopping Cart
- Responsive User Interface

## Tech Stack

- React.js
- JavaScript
- Tailwind CSS
- HTML5
- CSS3

## Project Status

Currently under active development.
